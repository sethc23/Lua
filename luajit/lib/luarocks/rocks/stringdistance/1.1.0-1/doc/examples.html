Examples

Here is an example that shows the basic use of the package. 

local d = require"stringdistance"

print("Testing levenshtein distance")

local s = "Soletrando"
local t = "soletranod"

-- Using levenshtein 
local dl = d.lev(s,t)
-- dl = 3

-- Using damerau-levenshtein 
local dd = d.dam(s,t)
-- dd = 2
